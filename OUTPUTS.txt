    because November stubble burning spikes pull the average up.
  - Q2 (Apr–Jun) is consistently the cleanest quarter.

OUTPUT 11 — output_11.png
Title    : Delhi AQI Forecasting — All Methods Monthly Avg (2015–2027) [5-panel]
Contents :
  5 individual forecast panels (SMA, SES, TES/Holt-Winters, ARIMA, ML Regression):
  Each shows: Historical (grey), Actual Test 2024 (green), Predicted (dashed),
              Forecast 2025–2027 (coloured line)

  SMA (6-month)           R2 = -0.670  Flat forecast ~260; misses seasonality
  SES (alpha=0.90)        R2 =  0.488  Flat ~330; more responsive but no cycles

  TES and ML Regression are the only methods that reproduce the annual
  winter-peak / monsoon-trough cycle in the forecast horizon.

OUTPUT 12 — output_12.png
Title    : Delhi AQI Forecasting — All Methods Monthly Avg (2015–2027) [7-panel]
Contents :
  Extended 7-panel version including Double Exponential Smoothing and
  Seasonal Decomposition + Trend method:

  SMA (6-month)              R2 = -0.670   Flat ~260; no seasonality
  SES (alpha=0.90)           R2 =  0.488   Flat ~330; no trend/season
  DES / Holt's Linear        R2 = -6.740   Diverges — linear trend extrapolation
                                            shoots upward, unsuitable for AQI
  TES / Holt-Winters         R2 =  0.994   Best classical method; full cycles
  ARIMA (3,1,2)              R2 = -0.540   Converges to flat ~300
  Seasonal Decomp + Trend    R2 =  0.974   Good seasonal structure, slight damping
  ML Regression Poly+Fourier R2 =  0.980   Best ML method; full cycles + trend

Key Observation:
  DES (Holt's Linear) fails badly on AQI data — its linear trend assumption
  extrapolates to unrealistic values. AQI is cyclical, not linearly growing.
  Seasonal Decomposition + Trend is a strong classical alternative to TES.

OUTPUT 15 — output_15.png
Title    : Dataset Fix: Official CPCB AQI Scale Applied
Contents :
  Side-by-side bar charts showing AQI bucket distribution:
  BEFORE Fix (random/incorrect labels) vs AFTER Fix (official CPCB scale)
  Official CPCB AQI Scale legend on the right

  After Fix distribution:
    Good=101, Satisfactory=602, Moderate=1294, Poor=675, Very Poor=781, Severe=200

Purpose:
  Documents a data preprocessing step where the original AQI_Bucket labels
  in the raw Kaggle dataset were re-assigned using the official CPCB breakpoints
  (0–50 Good, 51–100 Satisfactory, 101–200 Moderate, 201–300 Poor,
  301–400 Very Poor, 401–500 Severe) to ensure label correctness before
  any classification modelling.

Note:
  This is an important data quality step. The raw dataset had approximately
  uniform category counts, which is unrealistic for Delhi. After applying
  official breakpoints, Moderate becomes the dominant class (as expected
  from Delhi's actual air quality profile).

OUTPUT 16 — output_16.png
Title    : Delhi AQI Forecasting — All Methods (2015–2027) [Clean Version]
Contents :
  5-panel clean version of the forecasting comparison:
  SMA (6-month), SES, TES/Holt-Winters, ARIMA (3,1,2), ML Regression Poly+Fourier

  Each panel shows Train (black), Actual Test 2024 (green),
  Predicted (dashed), Forecast 2025–2027 (coloured line).

  ML Regression panel includes metrics: MAE=12.32, RMSE=14.3, R2=0.98

  Vertical red dotted line marks the train/test split boundary (Jan 2025).

Key Visual Takeaways:
  SMA    : Flat, lagged forecast — no seasonal cycles
  SES    : Flat at ~330 — reactive to last value but no cycles
  TES    : Full seasonal cycles reproduced correctly
  ARIMA  : Flat trend ~250-300; understates seasonal swings
  ML Reg : Full seasonal cycles with realistic amplitudes

This figure directly answers Research Question 3 — seasonal models (TES,
ML Regression) vastly outperform non-seasonal ones (SMA, SES, ARIMA) for
Delhi AQI forecasting.
