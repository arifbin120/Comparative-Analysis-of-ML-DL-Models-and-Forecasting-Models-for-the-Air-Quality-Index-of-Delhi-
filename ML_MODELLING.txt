================================================================================
I.  REGRESSION MODELS
================================================================================

The comparative analysis of Ridge Regression, Random Forest, Gradient Boosting,
and XGBoost revealed significant differences in model behavior and predictive
accuracy for AQI estimation in regression modeling.

The performance of the regression models was evaluated, and the results are
summarized as follows:

  • Ridge Regression achieved the highest performance with an R² score of
    0.9859, along with a low MAE of 10.86 and RMSE of 13.51. The
    cross-validation score (CV R² = 0.9868) further confirms its strong
    generalization ability. Performs well because it handles multicollinearity
    using regularization. Suitable when data has linear relationships.

  • Random Forest produced an R² of 0.9833, with MAE = 11.61 and RMSE = 14.68,
    indicating slightly lower accuracy compared to Ridge Regression. Captures
    non-linear relationships by combining multiple decision trees (ensemble
    learning).

  • Gradient Boosting demonstrated competitive performance with an R² of 0.9844,
    MAE = 11.40, and RMSE = 14.20, showing improved accuracy over Random Forest.
    Sequentially improving errors from previous models, focusing on hard-to-
    predict data points.

  • XGBoost achieved an R² of 0.9847, with MAE = 11.25 and RMSE = 14.08.
    Advanced boosting with regularization and optimization.

Overall, all models exhibited strong predictive performance, with Ridge
Regression slightly outperforming other models in terms of accuracy and error
minimization. On the other hand, ensemble-based models such as Random Forest,
Gradient Boosting, and XGBoost also demonstrated strong performance. These
models can capture complex, non-linear relationships in the data.

================================================================================
II.  CLASSIFICATION MODELS
================================================================================

Comparing models such as Logistic Regression, Random Forest, Gradient Boosting,
and XGBoost revealed differences in performance and prediction accuracy across
AQI categories. The performance of these models was analyzed in terms of
Accuracy, Macro F1 Score, Weighted F1 Score, and cross-validated accuracy. A
confusion matrix was also created for all models to evaluate their performance
in predicting each AQI category specified by CPCB: Good, Satisfactory,
Moderate, Poor, Very Poor, and Severe. The results are summarized as follows:

  • Logistic Regression achieved an accuracy of 0.8728, with a Macro F1 score
    of 0.8418 and a cross-validation accuracy of 0.8569, reflecting solid
    baseline performance for a linear classifier.

  • Random Forest produced an accuracy of 0.8632, with Macro F1 = 0.8121 and
    CV Accuracy = 0.8481. This is attributable to its ability to capture
    non-linear class boundaries by combining multiple decision trees through
    ensemble learning, though the marginal underperformance suggests the added
    complexity did not translate into gains for this dataset.

  • Gradient Boosting demonstrated competitive performance, with an accuracy of
    0.8851, Macro F1 of 0.8537, and CV Accuracy of 0.8525, improving on both
    Logistic Regression and Random Forest.

  • XGBoost achieved the highest classification performance, with an accuracy of
    0.8892, Macro F1 of 0.8654, and CV Accuracy of 0.8566, outperforming all
    other models.

================================================================================
SUMMARY
================================================================================

Regression Winner  : Ridge Regression
  → Best R² (0.9859), lowest MAE (10.86) and RMSE (13.51), highest CV R² (0.9868)
  → Linear model wins because PM2.5 and PM10 have near-perfect linear
    relationships with AQI (r > 0.97)

Classification Winner : XGBoost
  → Best Accuracy (0.8892), F1 Macro (0.8654), CV Accuracy (0.8566)
  → Advanced gradient boosting with regularization handles class imbalance
    (Good class = only 101 samples) better than other models

Key Observation:
  In both regression and classification tasks, simpler or well-regularized
  models (Ridge, XGBoost with L2) outperform purely tree-based ensembles
  (Random Forest), confirming that the linear structure of PM2.5/PM10 → AQI
  relationships dominates the feature space.
